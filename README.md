# AGROD
AI-powered smart agriculture drone using CNN for real-time crop monitoring and disease detection. Trained on ~4700 images with 70/15/15 split, the system enables targeted pesticide spraying, reducing chemical usage. Integrates edge computing for on-device inference and efficient field analysis.

# AI-Based Smart Agriculture Drone (AGROD)

CNN-based system for real-time crop monitoring and disease detection.

## Key Highlights
- Trained on ~4700 images (70/15/15 split)
- Real-time crop classification
- Targeted pesticide spraying

## Tech Stack
- Python
- CNN
- OpenCV

## Outcome
Improved precision agriculture by reducing unnecessary pesticide usage.
